# Changelog

All notable changes to this project will be documented in this file.

## 0.2.2

- enabled chat history by default for the current game session;
- kept chat history persistence across full restarts disabled by default;
- finalized clickable chat URL behavior in the in-game chat UI;
- cleaned up project naming, namespace, and repository structure for public release;
- prepared GitHub-friendly project metadata and documentation.

## 0.2.1

- cleaned up the plugin package and removed old probe/cosmetic leftovers;
- disabled chat history by default temporarily to stop broken history restore loops;
- simplified config and removed obsolete settings.

## 0.2.0

- renamed the plugin to `BetterCommunication`;
- added clickable Steam profile links for supported player names;
- added clickable Steam profile links for supported player avatars;
- added clickable URLs in in-game text chat;
- added local chat history restore support.
